# APIArsenal Command: /create-api-explorer

Create the API Explorer page.

This page allows users to inspect generated API endpoints.

Layout:

Left panel: endpoint list
Right panel: endpoint details

Endpoint details must include:

HTTP method
URL
Request parameters
Headers
Example request
Example response

Include:

Copy endpoint button
Try request button (placeholder)
Code snippet examples

Style inspiration:

Postman
Stripe API docs
Supabase API explorer
