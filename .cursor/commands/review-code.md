# APIArsenal Command: /review-code

Perform a code quality review.

**Check for:**
- Architecture issues
- Tenant safety violations
- Fat controllers
- Missing validation
- Improper responses
- Security issues
- Inconsistent naming
- Missing tests

**Verify:**
- `project_id` enforcement
- API key authentication safety
- Module boundaries respected
- Platform vs module separation

Suggest improvements if violations exist.
