# APIArsenal Command: /create-resource-builder

Create the Resource Builder UI.

This allows users to define API resources for their project.

The interface must include:

• resource name input
• fields list
• field type selector
• add/remove field buttons
• validation rules

Each field row should include:

Field name
Field type
Required toggle
Default value
Delete action

The UI should feel similar to a database schema editor.

Follow a clean developer tool layout.
