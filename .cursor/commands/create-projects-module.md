# APIArsenal Command: /create-projects-module

Create the Projects module UI.

Features:

• project list page
• project card or table layout
• create project button
• project details page

The projects page should include:

• page header
• create project action
• searchable list of projects
• project cards or table rows

Each project card should show:

• project name
• description
• number of resources
• last updated time
• quick actions

The page must include:

• empty state
• loading state
• pagination if needed
