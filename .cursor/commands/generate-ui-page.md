# APIArsenal Command: /generate-ui-page

Delegate UI work to the UI architect agent.

**The UI agent must:**
- Use Next.js
- Use Tailwind CSS
- Use shadcn components
- Maintain modern SaaS dashboard design

**Pages must include:**
- Loading states
- Error states
- Empty states
- Responsive layout

**Focus only on:** UI, layout, component design, UX consistency.

**Never implement backend logic.**
