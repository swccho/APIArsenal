# APIArsenal Command: /create-page-ui

Create a production-ready UI page for the specified feature.

Requirements:

• Follow APIArsenal UI design rules
• Use React + Inertia + TypeScript
• Use Tailwind styling
• Use Lucide icons
• Follow Stripe / Linear style

The page must include:

• page title
• structured layout
• actions
• empty states
• loading states
• responsive behavior

The page must feel like a real SaaS product page.
