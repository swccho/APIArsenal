# APIArsenal Command: /create-dashboard

Create a modern dashboard page for APIArsenal.

The dashboard must include:

• page header
• key metrics cards
• recent activity section
• project overview cards
• quick actions

Metrics examples:

Projects count
API requests
Active resources
Recent deployments

Design requirements:

• clean card layout
• good whitespace
• clear hierarchy
• responsive grid

Follow Stripe / Linear style.
